1. העלה את הקבצים ל-GitHub

   - חבר את הריפו
   - הוסף Environment Variable:
     OPENAI_API_KEY = המפתח החדש שלך

   https://YOUR-RENDER-URL.onrender.com/webhook
   Method = POST

אם חשפת API Key קודם, בטל אותו וצור מפתח חדש.
